INSTRUCTIONS POUR L'IMAGE

Pour utiliser votre photo du couple devant la CN Tower :

1. Placez votre image dans ce dossier (public/)
2. Nommez-la : couple-cn-tower.jpg
3. Formats supportés : .jpg, .jpeg, .png

L'image sera automatiquement utilisée comme arrière-plan de la section Hero.
Le texte est positionné en haut pour ne pas gêner les visages sur la photo.
